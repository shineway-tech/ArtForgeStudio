# Product Boundary

## Purpose

ArtForge Studio 是跨平台桌面 AI 美术生产客户端，提供从灵感浏览、提示词整理、图片生成到本地作品管理的一体化工作台。客户端负责交互、任务展示、本地作品交付和恢复；远端商业与生成状态由 ArtForge 服务端管理。

## Current client areas

- 欢迎与登录：产品入口、账号登录、协议确认和版本要求。
- 灵感广场：浏览随客户端分发的灵感素材并进入创作。
- 美术创作：选择创作类型、模型、画质、比例、张数和参考图，提交提示词或图片任务。
- 生成历史：显示排队、执行、部分成功、完成、失败和取消状态，支持结果查看、重试与删除。
- 我的资产：浏览已保存到本机的生成作品和展示元数据。
- 模型管理：展示服务端提供的模型目录、能力和当前计费信息。
- 积分与会员：展示余额、流水、积分包、会员方案、权益和支付状态。
- 通知：展示服务端通知、任务结果和账户事件。
- 设置与账号中心：管理界面偏好、作品路径、登录方式和设备会话。

## Server-authoritative data

以下事实以服务端响应为准，客户端不能自行创建或修改：

- 用户身份、登录会话、设备会话和协议接受记录。
- 会员套餐、有效期、画质权限和当前会员状态。
- 积分余额、预占、结算、返还和流水。
- 积分包、订单、支付状态和权益发放结果。
- 模型目录、模型能力、可用画质、操作价格和版本。
- 提示词任务、图片生成任务、任务结果和远端生命周期。

客户端展示服务端动态下发的模型、会员、积分包和计费信息。文档与客户端代码不把某个模型名称、价格、赠送额度、充值折扣或操作扣费视为长期常量。

## Local data

客户端负责保存在本机的数据包括：

- 已下载并通过完整性校验的作品文件。
- 作品展示所需的本地元数据。
- 提示词草稿、自定义提示词、语言、主题、字体、卡片样式等界面偏好。
- 设备标识和存放在系统安全凭据中的刷新会话。
- 未完成请求、下载、交付确认和支付同步所需的恢复记录。

本地缓存可以支持离线浏览和恢复，但不能覆盖服务端的账号、会员、积分、订单、模型或任务事实。

## Deliberate exclusions

当前客户端不包含或不再采用：

- 用户可编辑的 Provider Endpoint、供应商 API Key 和供应商模型连接测试。
- 客户端本地发放积分、直接修改余额或自行判断支付到账。
- 邀请返利等没有正式服务端契约的模拟业务数据。
- 服务端数据库、队列、支付回调、模型密钥和运维部署实现。
- 在客户端文档中固定易变的商业配置。
